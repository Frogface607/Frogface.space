## Описание

Создать страницу/инструмент в Frogface Studio для конвейерной генерации контента. Пользователь задаёт тему/идею → AI генерирует текст поста → промпт для картинки → готовый пакет для публикации.

## Контекст

Два бренда:
- MyReply: голос «Мы освобождаем ваше время», стиль дерзкий, без буллшита
- Edison Bar: тёплый, гастрономический, с юмором

Форматы: TG-посты, сторис, рилс-скрипты. Дизайн через Freepik API.
