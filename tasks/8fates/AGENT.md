# Cursor Agent: 8fates

## Проект
8fates — интерактивные AI-истории с моральными дилеммами.
Каждый день новая история, 7 точек выбора, AI генерит уникальную концовку.

## Репозиторий
`Frogface607/8fates` — Next.js, TypeScript, Supabase, Framer Motion, Vercel.

## Что делать
1. Прочитай задачи в этой папке (`tasks/8fates/*.md`)
2. Бери задачи со статусом `pending`
3. Выполни, обнови статус, закоммить

## Контекст
- Суть: ежедневная история → 7 выборов → AI-концовка → сравнение с другими
- Сброс историй: полночь по МСК
- Длительность: ~10 минут на историю
- Социалка: шеринг судеб, сравнение решений
