## Описание

Реализовать систему коммуникации между Moltbot (AI-ассистент через OpenRouter/ChatGPT) и Cursor (AI-кодер). Поток: Архитектор говорит голосом → Moltbot структурирует → создаёт задачу через API/файл → Cursor берёт задачу и выполняет.

## Контекст

Архитектор использует голосовой ввод для генерации идей. Moltbot (или ChatGPT) структурирует потоки в задачи. Cursor имеет подписку Ultra с большими лимитами на Claude Opus 4.6. Нужен мост между двумя AI-системами через git repo.

Варианты коммуникации:
1. Файлы в `tasks/` — Moltbot кладёт, Cursor читает
2. API endpoint — Moltbot отправляет через HTTP
3. GitHub Issues — автоматическое создание
4. Комбинация всех трёх
