# Cursor Agent: MyReply

## Проект
MyReply — AI-сервис автоответов на отзывы для малого бизнеса.
Платформы: 2Gis, Яндекс.Карты. Чек 490₽/мес. MRR 178K₽ → цель 500K₽.

## Репозиторий
Отдельный репо (если есть) или задачи связанные с лендингом/маркетингом MyReply.

## Что делать
1. Прочитай задачи в этой папке (`tasks/myreply/*.md`)
2. Бери задачи со статусом `pending`, начни с `critical`
3. Выполни работу
4. Обнови статус в файле: `**Статус:** done`
5. Добавь `## Результат` в файл
6. Закоммить и запуш

## Контекст
- ЦА: владельцы малого бизнеса (рестораны, отели, салоны)
- Оффер: AI отвечает на все отзывы за 490₽/мес
- Стадия: soft launch, 1 платящий клиент, цель 10
- Стиль бренда: дерзкий, без буллшита, tech
