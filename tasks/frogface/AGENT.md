# Cursor Agent: Frogface

## Проект
Frogface.space — персональная Life OS. Этот репозиторий.

## Стек
Next.js 16, TypeScript, Tailwind CSS 4. Деплой: Vercel.
AI: OpenClaw на VPS (primary), OpenRouter (fallback).
Storage: OpenClaw memory (KV), localStorage (cache).

## Что делать
1. Прочитай задачи в этой папке (`tasks/frogface/*.md`)
2. Бери `pending`, начни с `critical`
3. Код пиши тут в src/
4. Проверь `npx next build` перед коммитом
5. Обнови статус задачи, закоммить

## Правила
- TypeScript, английские имена переменных
- UI текст: русский
- Коммиты: feat:/fix:/chore: префиксы
- Тёмная тема, RPG-стиль
